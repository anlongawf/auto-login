# AutoLogin (Paper & Folia)

A high-performance Minecraft server plugin that automatically logs players in based on their previous session (IP and UUID match). Designed for modern server infrastructures, compatible with both standard **Paper** and multi-threaded **Folia**.

## 🚀 Features

*   **🧵 Multi-Threaded Support**: Absolute compatibility with Folia's region-based scheduling.
*   **⚡ Instant Authentication**: Uses optimal `AsyncPlayerPreLogin` caching for 0.00ms login delays.
*   **🛡️ Discord Webhook Notification**: Real-time alerts for automatic logins or suspected overlaps.
*   **🗄️ SQL Syncing**: Built-in support for lightweight SQLite or concurrent MySQL for proxy syncing.
*   **👑 Admin Bypass**: Advanced permission node `autologin.bypass` enforces manual pins for security.
